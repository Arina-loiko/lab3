**Блоки кода**
Встроенный (inline) код:
Команда `git status` показывает состояние репозитория.
Многострочный код: 
```csharp
print("Hello, Markdown!")
```
Можно указывать язык: python, js, bash, csharp и др.

Первый уровень
>> Второй уровень
